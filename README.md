# Building Evacuation Simulation (Multi-Agent System)

The project models crowd dynamics during emergency evacuation using
**multi-agent systems**, combining physical movement models with
behavioral decision-making.

The goal is to analyze how **building layout, agent behavior, and
environmental hazards (e.g. smoke)** affect the **total evacuation time
and congestion patterns**.

------------------------------------------------------------------------

# Project Goals

The main objectives of this project are:

-   simulate evacuation scenarios in complex indoor environments
-   model **individual agents with heterogeneous behaviors**
-   analyze **crowd dynamics and bottlenecks**
-   evaluate how **architectural design and behavioral parameters**
    influence evacuation efficiency
-   perform **statistical analysis of simulation outcomes**

------------------------------------------------------------------------

# Key Research Questions

The simulation investigates:

-   How does **exit width** influence evacuation time?
-   How does **crowd density** affect congestion formation?
-   What is the impact of **reaction time distribution**?
-   How does **limited visibility (smoke)** change evacuation routes?
-   Can **leader agents** improve evacuation efficiency?
